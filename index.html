Hierarquização de Texto

(Voltar)

Todo conteúdo produzido para si mesmo e para terceiros, precisa seguir algumas normas e semânticas. Com HTML não poderia ser diferente. A hierarquização de texto representa uma meneira de organizar visualmente o site, alocando o nome do título daquele bloco e facilitando a compreensão e leitura do usuário do nosso site.

O que são Tags?

(Voltar)

Tags são uma maneira de informar ao navegador que está sendo utilizado pelo usuário a forma que o site está estruturado. É como se o HTML utilizasse as Tags para informar ao navegador como ele deve mostrar a aparência do site a quem o está utilizando.

O que são Atributos?

(Voltar)

Atributos são as características das Tags. São os atributos que poderão definir a identidade daquela tag (como o atributo "id", por exemplo) ou a classe que ela vai fazer parte em um arquivo CSS (como "class"), dentre outras.

A importância dos links

(Voltar)

Os links são importantíssimos para uma página HTML, tendo em vista que eles irão promover a navegabilidade do site. Links nada mais são do que "linkar", articular uma página à outra página, dentro do mesmo site. Ele incentiva a permanência do usuário no nosso site e nos ajuda a deixar a experiência do usuário mais leve e intuitiva.
